## Propósito del Informe

Este informe proporciona un análisis detallado de cuatro tiendas con el objetivo de identificar la que posee el mayor potencial de venta y representa la mejor oportunidad de inversión para el Sr. Juan. La recomendación se basa en una evaluación exhaustiva de métricas clave, que incluyen:

*   **Rendimiento Financiero:** Análisis de los ingresos totales de cada tienda.
*   **Popularidad de Productos:** Evaluación de las ventas por categoría y la identificación de productos más y menos vendidos.
*   **Satisfacción del Cliente:** Cálculo de las calificaciones promedio de los clientes.
*   **Eficiencia Operativa:** Determinación del costo de envío promedio por tienda.
*   **Distribución Geográfica de Ventas:** Mapeo de las ventas para analizar su distribución espacial y los patrones regionales.

Cada una de estas métricas es fundamental para una toma de decisiones sólida, asegurando la viabilidad económica, la lealtad del cliente, la eficiencia del inventario y la optimización de los márgenes de ganancia.

## Análisis Clave Realizados

El análisis del notebook se estructura en las siguientes secciones:

1.  **Análisis de Facturación:** Cálculo de los ingresos totales de cada tienda para evaluar su desempeño financiero.
2.  **Ventas por Categoría:** Identificación de la cantidad de productos vendidos por categoría, tanto en conteo absoluto como en porcentaje, para determinar las categorías más y menos populares.
3.  **Calificación Promedio de la Tienda:** Evaluación de la satisfacción del cliente a través de las calificaciones promedio recibidas por cada tienda.
4.  **Productos Más y Menos Vendidos:** Identificación de los productos individuales con mayor y menor volumen de ventas en cada tienda.
5.  **Envío Promedio por Tienda:** Cálculo del costo de envío promedio para cada tienda para comprender el impacto logístico y el costo adicional para el cliente.
6.  **Análisis del Desempeño Geográfico:** Utilización de datos de latitud y longitud para mapear las ventas, identificar concentraciones geográficas de ventas (puntos calientes) y analizar patrones regionales.

## Librerías de Python Utilizadas

Este notebook utiliza las siguientes librerías de Python para el análisis y la visualización de datos:

*   `pandas`: Para manipulacián y análisis de datos.
